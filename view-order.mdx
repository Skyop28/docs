After placing an order, you can track its progress through your dashboard. This page explains how to access order details and understand the information displayed.

## Finding your order

There are several ways to access your order details:

**From the confirmation page**: After completing payment, the confirmation page displays your order details and delivery status. Bookmark this page or note the order reference number for future access.

**From the dashboard**: Navigate to **Orders** in the sidebar to see all your orders. Click any order to view its details.

**From email**: Order confirmation emails include a link to view your order details directly.

**Using the order reference**: If you have your order reference number (formatted as `ORD-XXXXXXXXXX`), you can search for it in the Orders section.

## Order details page

[IMAGE PLACEHOLDER: Order details view]

The order details page provides comprehensive information about your purchase.

### Order summary

The top section displays key order information:

- **Reference number**: Unique identifier for this order
- **Status**: Current state of the order
- **Pokecoin amount**: Quantity purchased
- **Total paid**: Payment amount and currency
- **Order date**: When the order was placed

### Delivery information

Details about where and when your Pokecoins are delivered:

- **Market ID**: The destination for your Pokecoins
- **Delivery status**: Whether delivery is pending, in progress, or complete
- **Delivery timestamp**: When the Pokecoins were sent (for completed deliveries)

### Payment information

Transaction details from the payment:

- **Payment method**: How you paid
- **Payment status**: Whether payment was received
- **Transaction reference**: Identifier from the payment processor

### Order timeline

A chronological list of all events related to your order:

- Order placed
- Payment initiated
- Payment confirmed
- Processing started
- Delivery attempted
- Delivery confirmed

Each event includes a timestamp showing exactly when it occurred. This timeline is useful for understanding the order lifecycle and identifying any delays.

For instant orders, you will typically see the order progress from "Processing" to "Delivered" within seconds after payment confirmation.

## Order actions

Depending on the order status, different actions are available:

**Pending orders**:
- Complete payment (if not finished)
- Cancel order (before payment)

**Delivered orders**:
- View delivery confirmation
- Report an issue

**Failed orders**:
- View failure reason
- Retry order
- Contact support

## Sharing order information

If you need to share order information with support or for your records:

- Copy the order reference number for quick identification
- Use the print or export option to save order details
- Take a screenshot of the order timeline for complete documentation
